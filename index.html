第三方工具集

mocky.io
神器。多人协作开发时，当需要开发某个客户端，但是服务端 API 尚不具备时，可以用这个工具先造一些假的 HTTP 请求地址，填好预期的 response 数据。让客户端开发者先用着，最后集成时再换成真实的服务端 API。本来打算自己开发这样的东西，但是发现已经有人做过了，很不错。
http://www.mocky.io/
